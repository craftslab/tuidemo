## Introduction

*tuidemo* is the demo of [tui-rs](https://github.com/fdehau/tui-rs) written in Rust.



## Prerequisites

- Rust >= 1.59.0



## Run

```bash
make install
make build
./target/x86_64-unknown-linux-musl/release/tuidemo"
```



## Docker

```bash
make docker
docker run ghcr.io/craftslab/tuidemo:latest
```
